# Investigations Index

Standalone deep-dive analyses spawned by the weekly digest pipeline when a signal warrants extended investigation.

Each investigation lives in `investigations/YYYY-WW-<slug>/` and contains:
- `hypothesis.md` — expected signal, falsification criterion, data sources
- `analysis.py` — computational analysis (optional)
- `findings.md` — results and conclusions
